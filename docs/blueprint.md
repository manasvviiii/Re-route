# **App Name**: PivotPath

## Core Features:

- Empowerment Hub: Provide a curated selection of motivational articles and success stories.
- Engineering Domain Navigator: Offer detailed guides, job outlooks, and skill requirements for various engineering domains.
- Aptitude AI Tool: Administer a compatibility test that leverages AI as a tool to align user traits with potential engineering paths.
- Personalized Roadmap: Generate personalized action plans with compatible career options based on test results.
- Curated Learning Resources: Aggregate resources to educational content for upskilling.
- Community Forum: Enable a supportive online forum for users to share experiences, seek advice, and connect with peers.

## Style Guidelines:

- Primary color: Muted teal (#66A3A3) for calm and growth.
- Background color: Creamy off-white (#FAF9F6) to reduce eye strain.
- Accent color: Warm sunset gradient (soft orange-to-yellow), transitioning from #FFB347 to #FFDA63, for hope and energy.
- Font: 'Figtree' sans-serif for a friendly and approachable feel.
- Use custom icons representing various engineering fields (e.g., data science, AI, electrical) using line art.
- Clean, card-based scrolling layout to organize content into digestible blocks.
- Subtle animations and transitions to maintain user engagement.
